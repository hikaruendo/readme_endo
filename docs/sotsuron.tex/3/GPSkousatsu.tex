\begin{comment}
実験結果をうけて，GPSの誤差原因について調べるため追実験を行った．先行研
究では3号棟周辺でのデータ取得の際に，本実験で得られたような
ランダムノイズの影響したようなデータは得られていなかった．そこで，Kinect
を搭載，使用することによってノイズが生じた可能性を考え検証を行った．
図\ref{Kinectを搭載しない場合}，図\ref{Kinectを搭載し使用しない場合}
はそれぞれKinectを搭載しない場合とKinectを搭載し使用しない場
合の，GPSデータ取得結果である．経路は本実験を行った経路と同じ経路を本実
験と連続して走行させた．\par
結果Kinectを使用しなかった2つの追実験では，ランダムノイズの影響したよう
な経路はみられなかった．よって，Kinectを使用することによる信号や電波の
干渉が原因であると推測する．
