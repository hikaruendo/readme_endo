本研究のKinectセンサを用いたオンラインSLAMでは，KinectセンサからRGB-Dデー
タを取得し，並行して$3$次元点群の位置合わせを行い，自己位置の推定，補正
を行う，また同時に点群で構成される環境地図を構築する．このオンラインSLAMには，
RTAB-Map(Real-Time Appearance-Baced Mapping)というグローバルベイジアンルー
プ閉鎖検出に基づいたRGB-DグラフSLAMアプローチの手法を用いている．この手法で
の処理を概括的に以下に示す．\par
\par
・ RGB-Dデータ取得\par
・ SIFT特徴量抽出\par
・ 自己姿勢推定\par
・ 環境地図構築\par
・ ループ閉鎖検出\par
\par
この手法では，取得した特徴点をランドマークとして保持し，ループ閉鎖検出に
より再度同じ特徴点を取得するとループの閉じ込みを行い，自己姿勢推定にフィー
ドバック補正をかけることができる．
オンラインでのループ閉鎖検出は，クラスタ半径1m，クラスタ角度30度，反復回数5回
で行われる．オフラインでは最大で，クラスタ半径99m，クラスタ角度180度，反
復回数100回で検出を行うことができる．\par
以下に実際にループ閉鎖検出を行った際の、ループ閉鎖検出前とループ閉鎖検出
後の図をそれぞれ図\ref{kinect環境地図ループ閉鎖検出前}と図\ref{kinect環
境地図ループ閉鎖検出後}に示す．
図\ref{kinect環境地図ループ閉鎖検出後}ではクラスタ半径を15mに広げてルー
プ閉鎖再検出を行った．上部に見られた環境地図のずれを検出範囲を変えた再検
出によって補正し環境地図を再取得できた．これに伴い，出力された各点の自己位置の補正も行われている．
